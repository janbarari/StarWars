# StarWars
 ![](starwars.gif)

This project gets information form an api and uses the following technologies:

  - Kotlin
  - Clean Architecture
  - Design Patterns (Singleton, Factory, Repository, Strategy, Observer)
  - UI Trasition
  - MVVM
  - RxJava
  - Retrofit
  - Android Navigation Components
  - LiveData
  - ViewModel
  - Kodein (Dependency Injector)
  - Firebase
